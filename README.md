# DUET+
同志社大学の学生用ポータルサイトDUETに、より詳細な情報を表示させます。  

今実装している機能  
- 自身の成績を確認できるページに、各授業の成績分布情報や、同じ授業を受けた生徒の目安GPAなどを表示します。

# build & publish
- `yarn run build:extension`

## install
パソコン向けにはChrome、スマホ向けにはiOSのSafariの拡張機能として公開しています

Chrome: https://chromewebstore.google.com/detail/mofjbejpdfdfkbiicjhimkodkijekjdk?utm_source=item-share-cb

Safari(iOS): https://apps.apple.com/jp/app/duet/id6743043491
